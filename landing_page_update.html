Get Started with FlareGuard

Follow these steps to start securing your Cloudflare configuration:

  1. Choose an audit type:
    • Zone Security Audit - Audit SSL/TLS, WAF, and other zone settings
    • Zero Trust Audit - Evaluate Access applications and Zero Trust configuration
    • Multi-Zone Audit - Audit multiple zones at once
  2. Create an API token:
    • Go to your Cloudflare dashboard
    • Create a new API token with appropriate read permissions
    • For Zone Security: Zone Read, SSL and Certificates Read, WAF Read, DNS Read
    • For Zero Trust: Account Access: Apps and Policies Read, Gateway Read
  3. Run your first audit:
    • Enter your Zone ID or Account ID (depending on audit type)
    • Paste your API token
    • Click "Run Audit" to get your security assessment
